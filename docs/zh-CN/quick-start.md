# 快速开始

## 1. 启动依赖

在项目根目录执行：

```bash
docker compose up -d postgres redis minio
```

看到 PostgreSQL、Redis、MinIO 容器启动后继续。

## 2. 启动后端和前端

后端启动后访问 `http://localhost:8080/swagger-ui.html` 可查看接口。

前端启动后打开 `http://localhost:8000`。

## 3. 检查流程资产

进入「流程模型」，确认已有可部署的协同审批流程。

如果开发环境缺少流程资产，可打开隐藏的「开发资产检查」页面，点击「补齐流程资产」。该操作会准备：

- 协同审批流程 `collaborativeApproval`
- 业务申请表
- 发起表单和多人会签任务节点的表单绑定

重复点击不会生成大量重复数据。

## 4. 发起业务流程

进入「发起流程」。

选择「协同审批流程」，填写事项名称、事项说明、期望结果、金额和多个审批人，点击「发起」。申请人和所属部门由当前登录成员自动带出。

## 5. 处理待办

进入「组织权限」确认审批人成员已启用，再进入「我的待办」打开「多人会签」待办。

填写审批结论和处理意见，点击「完成任务」。

每个审批人都处理完成后，流程结束。

## 6. 查看结果

进入「我的申请」打开申请详情。

应能看到：

- 实例状态
- 审批进度
- 当前任务
- 各审批人的处理意见
- 表单快照
- 执行轨迹
- 审计记录

管理员进入「审计日志」可查看关键操作记录。
