# 快速开始

## 1. 启动依赖

在项目根目录执行：

```bash
docker compose up -d postgres redis minio
```

看到 PostgreSQL、Redis、MinIO 容器启动后继续。

## 2. 启动后端和前端

后端启动后访问 `http://localhost:8080/swagger-ui.html` 可查看接口。

前端启动后打开 `http://localhost:8000`。

## 3. 检查流程资产

进入「流程设计」或「流程模型」，确认已有可部署的采购申请流程。

如果开发环境缺少流程资产，可打开隐藏的「资产检查」页面，点击「补齐流程资产」。该操作会准备：

- 采购申请流程 `purchaseApproval`
- 采购申请单
- 部门审批和财务审批任务节点的表单绑定

重复点击不会生成大量重复数据。

## 4. 发起采购申请

进入「流程实例」。

选择「采购申请」，填写申请人、申请部门、采购事项、预算金额、申请事由、部门审批人和财务审批人，点击「启动流程」。

## 5. 处理待办

进入「任务中心」，切换到部门审批人账号，打开「部门审批」待办。

选择审批结论，填写审批意见，点击「完成任务」。

再切换到财务审批人账号，处理「财务审批」待办。

## 6. 查看结果

进入「流程实例」打开实例详情。

应能看到：

- 实例状态
- 审批进度
- 当前任务
- 部门审批和财务审批意见
- 表单快照
- 执行轨迹
- 审计记录

进入「审计日志」可查看关键操作记录。
