# 表单指南

## 创建表单

进入「表单管理」。

填写：

- 表单标识
- 表单名称
- 字段名称
- 业务字段
- 字段类型
- 控件类型
- 是否必填

点击「创建」。

## 业务申请表字段

业务申请表通常包含：

- 申请人
- 所属部门
- 事项名称
- 事项说明
- 期望结果
- 金额
- 审批人
- 备注

## 绑定表单

进入「表单绑定」。

选择已部署模型，分别选择「发起表单」或「多人会签」任务节点，选择「业务申请表」，点击「绑定」。

绑定后，任务详情页会显示中文表单。

## 表单快照

完成任务时，系统保存表单数据和表单结构快照。之后即使表单升级，历史任务仍能查看当时填写的数据。
