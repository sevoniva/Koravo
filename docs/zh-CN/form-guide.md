# 表单指南

## 创建表单

进入「表单管理」。

填写：

- 表单 Key
- 表单名称
- Schema JSON
- UI Schema JSON

点击「创建」。

## 请假申请表字段

内置请假申请表包含：

- 申请人
- 请假类型
- 开始日期
- 结束日期
- 请假天数
- 请假原因
- 附件说明

## 绑定表单

进入「表单绑定」。

选择已部署模型，任务节点 Key 填 `approveTask`，选择「请假申请表」，点击「绑定」。

绑定后，任务详情页会显示中文表单。

## 表单快照

完成任务时，系统保存表单数据和表单 schema 快照。之后即使表单升级，历史任务仍能查看当时填写的数据。
