# 表单指南

## 创建表单

进入「表单管理」。

填写：

- 表单编码
- 表单名称
- 字段名称
- 字段编码
- 字段类型
- 控件类型
- 是否必填

点击「创建」。

## 采购申请单字段

采购申请单通常包含：

- 申请人
- 申请部门
- 采购事项
- 预算金额
- 申请事由
- 部门审批人
- 财务审批人
- 备注

## 绑定表单

进入「表单绑定」。

选择已部署模型，选择「部门审批」或「财务审批」任务节点，选择「采购申请单」，点击「绑定」。

绑定后，任务详情页会显示中文表单。

## 表单快照

完成任务时，系统保存表单数据和表单 schema 快照。之后即使表单升级，历史任务仍能查看当时填写的数据。
