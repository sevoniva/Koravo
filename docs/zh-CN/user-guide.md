# 用户指南

## 首页

首页显示后端状态、租户、用户、流程配置状态、任务数量和 HTTP 连接器摘要。

点击「初始化配置」可准备请假审批流程。

## 快速开始

按页面步骤操作即可跑通第一个流程。每一步都有跳转按钮。

## 系统设置

设置开发期请求头：

- `X-Tenant-Id`：默认 `default`
- `X-User-Id`：默认 `admin`
- `X-Request-Id`：可选，不填则后端自动生成

这些值会随前端请求发送到后端。

## 审计日志

进入「审计日志」，可按用户、动作、资源类型、资源 ID、请求 ID 和时间查询。

关键操作会记录审计，包括流程配置初始化、流程启动、任务完成、表单保存、表单绑定、数据源测试和连接器调用。
