# 用户指南

## 首页

首页显示服务状态、流程模型、运行实例、待办任务、连接器调用和最近操作。

点击「创建流程」进入流程设计器，从流程建模开始配置真实业务流程。

## 业务流程

建议按以下顺序跑通一条采购申请流程：

1. 进入「流程设计」，创建或编辑采购申请流程。
2. 进入「表单管理」，配置采购申请单字段。
3. 进入「表单绑定」，把审批任务节点绑定到表单版本。
4. 进入「流程模型」，校验并部署流程。
5. 进入「流程实例」，发起采购申请。
6. 进入「任务中心」，由部门审批人和财务审批人分别处理待办。
7. 回到「流程实例」查看进度、审批意见、表单快照和审计记录。

「资产检查」仅用于开发期补齐缺失流程资产，不作为日常业务入口。

## 系统设置

设置开发期请求头：

- `X-Tenant-Id`：默认 `default`
- `X-User-Id`：默认 `admin`
- `X-Request-Id`：可选，不填则后端自动生成

这些值会随前端请求发送到后端。

## 审计日志

进入「审计日志」，可按用户、动作、资源类型、资源 ID、请求 ID 和时间查询。

关键操作会记录审计，包括流程模型保存、流程部署、流程启动、任务完成、表单保存、表单绑定、数据源测试和连接器调用。
